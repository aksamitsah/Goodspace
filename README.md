## 🤖 Installation:

- open Goodspace.xcodeproj
- Goto Target `Goodspace` -> `SignIn & Compatability` select `team` & change  `package name`
- If you issue on SPM Goto `File` -> `Packages` -> `Reset Packages Cache`
- Ready To Go...
